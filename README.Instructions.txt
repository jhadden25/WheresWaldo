Where's Waldo -- TensorFlow 2 Custom Object Detection Model
---------------------------------------------------

To use this repository follow the instructions below.

FOR TRAINING YOUR OWN MODEL:

Download and run the TrainingTF2.ipynb file on Google Colab as is. 
The default configuration is set to 2000 training steps with a batch size of 16.
The completed model with these settings is saved within this repository.
It is reccommended that you enable GPU hardware acceleration on Google Colab to ensure efficiency.

----------------------------------------------------------

FOR TESTING OUR MODEL:

Simply download the TestingWaldo.ipynb file and after running the first code snippet:
	-Refresh the project hierarchy.
	-Move each folder within the WheresWaldo folder out into the working directory.
